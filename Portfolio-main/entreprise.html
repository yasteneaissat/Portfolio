Le Système d'Information (SI)

Environnement technique de développement :

  • Serveurs de test : Localhost via XAMPP / WAMP
  • Langages : HTML5, CSS3, JavaScript pour le Front-end
  • Back-end : PHP (Architecture modulaire avec include)
  • Base de données : MySQL gérée via phpMyAdmin
